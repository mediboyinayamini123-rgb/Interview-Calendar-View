# Interview Calendar View

## Project Overview

Interview Calendar View is a frontend web application developed using React.js and FullCalendar. The application displays scheduled interviews in an interactive calendar format with filtering functionality and automatic interview status updates.

---

# Features

* Interactive monthly calendar
* Scheduled interview cards
* Interview filtering
* Automatic Scheduled and Completed status
* Month navigation
* Responsive user interface
* Mock interview schedules for multiple months

---

# Technologies Used

* React.js
* Vite
* JavaScript
* CSS
* FullCalendar

---

# Libraries Used

* @fullcalendar/react
* @fullcalendar/daygrid
